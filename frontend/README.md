# Smartfill AI – Frontend

Modern React-based user interface for uploading PDFs, providing natural language prompts, and generating filled documents using AI.
 
 
## ✨ Features
 
- Clean, responsive dark/light theme UI with Tailwind CSS
- PDF file upload with drag & drop support
- Natural language prompt input for document filling instructions
- Real-time status feedback (uploading, processing, success/error)
- Automatic download of AI-filled PDF
- Beautiful animations & modern glassmorphism design
- Mobile-friendly layout
 
## 🛠️ Tech Stack
 
- **Framework**: React 18 + Vite
- **Styling**: Tailwind CSS v3 / v4
- **State Management**: React Hooks (`useState`)
- **File Handling**: Native `<input type="file">` + FormData
- **HTTP Client**: Fetch API
- **Build Tool**: Vite
